# Changelog

## 0.2.0 — 2026-05-15

Refocused on producing a BRS artifact only.

- Dropped Phase 2: no longer hands off to `write-a-skill` or generates any skill.
- Output is now `BRS-<slug>.md` plus optional supporting files: `GLOSSARY.md`, `STAKEHOLDERS.md`, `SCENARIOS.md`, `OPEN-QUESTIONS.md`, `OUT-OF-SCOPE.md`.
- Eight BRS questions now lead with **Goal** and add **Constraints** (replacing inputs/outputs, which were implementation-flavoured).
- Adds "done-when" declaration and explicit escalation suggestion to `/frs-session` when system behaviour is the natural next step.
- Templates for each supporting file included inline.

## 0.1.0 — 2026-05-15

Initial release (superseded).

- Two-phase session: BRS interview then hand-off to `write-a-skill` to scaffold a skill.
- Eight-question BRS interview.
